01  North Star
canon is a taste intelligence platform. It starts with film and expands across all creative domains — restaurants, music, books, travel. The core promise is not "what to watch" but "who you are as a viewer, and what that means for every aesthetic choice you make."
"Letterboxd tells you what you watched. Canon tells you who you are."
The product is opinionated, precise, and personal. Every score is user-authored. The system reveals patterns in what you've already decided, rather than deciding for you. It should feel like something you'd want to send to friends — a taste identity, not a database.

Brand name rationale:
"Make it canon" — already in the cultural zeitgeist. Canon owns the idea that your taste is definitive, not provisional. Works across every domain. "Develop your palate" is the supplementary voice inside the product — warm, equalizing, personal.

02  Design System
Typography
• Display: Playfair Display — serif, italic, heavy. Titles, scores, big moments.
• Body: DM Sans — clean, readable. All prose and UI labels.
• Data/mono: DM Mono — monospace. All metadata, stats, tags, codes.

Palette
#F4EFE6   paper — background
#0C0B09   ink — primary text
#3D5A80   slate blue — interactive, data highlights, CTAs
#E8623A   persimmon — primary CTA only (use sparingly)
#837A6A   dim — secondary text, labels
#D8D0C0   rule — borders, dividers
#FAF6F0   card-bg — card surfaces

Design Principles
• Swiss-editorial. Typographic-first. Feels like a tool a film critic would actually use.
• No dark mode yet. Clean paper aesthetic is intentional and differentiating.
• Animations: subtle fadeIn only. No bounces, no flashy transitions.
• Mobile: currently desktop-first. Responsive pass is Phase 4.

04  React: The Decision
Start with Vite + vanilla JS, migrate to React in Phase 3 when the friends layer lands.
Why not React now: The app is data-rendering with light interactivity. Vanilla JS handles it fine. React's component model only pays off when many interactive stateful components need to re-render independently.
Why we'll want React later: Friends layer introduces real-time updates (friend activity, shared lists, group views). At that point, managing UI state manually becomes painful.
Decision: Build Phase 0–2 in vanilla JS. Reassess at Phase 3 kickoff.

05  Data Model
Supabase: ledger_users table
id                  uuid primary key
username            text unique  e.g. "alex-7742"
display_name        text
archetype           text  e.g. "Narrativist"
archetype_secondary text
weights             jsonb  { plot, execution, acting, production,
                            enjoyability, rewatchability, ending, uniqueness }
harmony_sensitivity numeric 0.0–1.0
movies              jsonb  [ array of film objects ]
created_at / updated_at  timestamp

Film Object Schema
title               string
year                number
total               number (recalculated on weight change — never trust cached)
director            string comma-separated
writer              string comma-separated
cast                string comma-separated (top 8 from TMDB)
productionCompanies string comma-separated
poster              string  TMDB path e.g. "/abc123.jpg"
overview            string  synopsis from TMDB
scores              object  { plot, execution, acting, production,
                             enjoyability, rewatchability, ending, uniqueness }

Scoring Formula
total = Σ(score[cat] × weight[cat]) / Σ(weight[cat])
Weights are user-specific. recalcAllTotals() must be called after any weight change.

9 Archetypes
Visceralist · Formalist · Narrativist · Humanist · Completionist · Sensualist · Revisionist · Absolutist · Atmospherist
Each has a distinct weight distribution. Derived from 6-question onboarding quiz. Editable post-onboarding via archetype modal (click username in masthead).

06  Screens & Features (Current)
Cold Landing
Full-screen overlay for new/logged-out users. Headline: "Make it canon." Supporting copy: "Everyone has a palate. The films you love are trying to tell you something about yourself. Canon listens — and shows you the pattern." Single persimmon CTA: "Discover your taste →". Footer note: "Starting with film. Expanding to everything."

Onboarding (3 entry points)
New user: cold landing → name → poetic intro + 6-question quiz → archetype reveal (name + description + quote + secondary + username — save it!) → Enter canon.
Returning user: username lookup → fetches profile + films from Supabase → loads directly.
Import: drag-and-drop film_rankings.json → validates → loads MOVIES → proceeds to quiz.

Rankings
Sortable table — all 8 category columns + total. Click any header to sort asc/desc. Click any row to open film modal. Rank shown based on total score regardless of current sort.

Film Modal
TMDB poster · synopsis · rank #X of N · clickable director/writer/cast/company chips (navigate to Explore) · per-category scores + rank + weight bars · inline score editing (Edit scores button → sliders with live score + label updates → Save) · nearby films in rankings (clickable).

Explore
Index tabs: Directors / Writers / Actors / Companies. Shows entities with 2+ films, sorted by avg score. Each card shows rank among peers (#3 of 24). Entity view: avg score, rank among peers, film count, per-category breakdown with per-category rank, insight blurb, full film list.

Predict
Search unrated films. Builds taste fingerprint (per-category mean/std/range, top 10, bottom 5, weights, archetype). Finds comparable films by shared director/cast. Calls Claude via Cloudflare proxy. Returns: predicted score per category, confidence level, reasoning in 2nd person ("you") anchored to user's specific films. "Add to list" button pre-fills Add Film search.

Analysis
Top directors, actors, genre patterns, score distributions. Rendered dynamically from MOVIES array.

Calibrate
Elo-based head-to-head system. Choose category + intensity (15/30/50 matchups). Review proposed changes with checkboxes. Apply writes scores + recalcs totals + syncs.

Add Film
4-step: TMDB search → cast/crew/company curation → 8-category scoring (sliders + head-to-head comparison) → result. Poster and overview saved on film push.

Profile Screen
Nav screen. Sections: archetype deep-dive (name, description, quote, secondary), taste fingerprint (radar chart of weights vs archetype baseline + avg score bars per category + key stats), signature films (top 5 by total, clickable), Canon Card (shareable screenshot card with archetype description). Sign out at bottom.

Archetype / Weight Editor Modal
Accessible via "Edit weights →" in Profile screen. Weight inputs with live bar preview. Save recalcs all totals + syncs. Sign out also available here.

Masthead
Clicking display name navigates to Profile screen.

Empty State (Rankings)
When 0 films: 60vh centered CTA — "Start with one you love." + persimmon "Rate your first film →" button. Column headers hidden.
